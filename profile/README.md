# Open Tiger

## 简体中文

Open Tiger 是一个微服务开发平台，面向企业级应用的研发、集成、部署和运维场景。

平台围绕微服务工程实践提供基础能力，包括：

- 网关与统一入口
- 认证授权与用户体系
- 系统管理与基础配置
- 工作流与任务调度
- 公共组件与 API 契约
- 前端管理端与本地 Docker 部署支持

Open Tiger 的目标是沉淀一套可复用的微服务研发基础设施，让团队可以更快完成服务开发、联调、部署和交付。

## English

Open Tiger is a microservice development platform for enterprise application development, integration, deployment, and operations.

It provides foundational capabilities for microservice engineering, including:

- API gateway and unified entry point
- Authentication, authorization, and user management
- System administration and base configuration
- Workflow and job scheduling
- Shared components and API contracts
- Web management console and local Docker deployment support

Open Tiger aims to provide reusable microservice infrastructure so teams can build, integrate, deploy, and deliver services more efficiently.
